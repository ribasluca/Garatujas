RESUMO DE HTML

e como se fosse a base de um site tipo o esqueleto de uma pagina na internet
e com ele que voce organiza tudo que aparece na tela como titulos,texto,imagens e links
ele funciona usando "etiquetas"(tags) que dizem pro navegador o que cada coisa e
mesmo sendo simples ele e muito importante porque sem o HTML nao existiria site ele 
e o primeiro passo para comecar a criar qualquer coisa na web.voce pode usar uma tag para dizer que algo e um titulo ou para indicar que e um paragrafo 

RESUMO DE CSS

e uma linguage responsavel por definir a aparencia dos elementos do HTML,controlando cores,fonte,tamanhos,espacamentoe e 
layot.ele permite separar oconteudo da apresentacao,oque resulta em uma aparenia simples.a sintaxe de css eimporta por seletores e declaracoes,onde os seletores indicam
quais sao os elementos que serao estilizados e as declaracoes definem quais propriedades e valores sao aplicados 

 os seletores pode ser de diferentes tipos,como elementos,classe,id,atributo e pseudo-classes,pirmitindo selecionar elementos de forma
 especifica.O CSS segue uma logica em cascata baseada na hierarquia do HTML, possibilitando aplicar estilo de acordo com a posicao 
 dos elementos,como elementos descendentes ou filhos diretos,tornando a estilizacao mais precisa.
